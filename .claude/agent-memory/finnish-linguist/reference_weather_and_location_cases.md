Verified while authoring levels 11–12 (city/shopping + health/weather). Complements
[[predicative-case]].

**Impersonal weather → `on` + PARTITIVE adjective (no subject):**
- `On pilvistä` (облачно), `On aurinkoista` (солнечно), `Tänään on lämmintä`.
- Contrast a COUNTABLE subject which stays NOMINATIVE: `Kesä on lämmin`, `Sää on kylmä`,
  `Pää on kipeä`. Good wrong to encode: nominative where impersonal partitive is needed
  (`on aurinkoinen` → should be `aurinkoista`) and vice-versa (`kesä on lämmintä`).

**Place nouns that take ADESSIVE (-lla/-llä) for location, NOT inessive** (so direction
is -lle, not illative): `tori`→torilla/torille, `kassa`→kassalla/kassalle,
`lentokenttä`→lentokentällä (already in seed), `asema`→asemalla, `katu`→kadulla.
But normal buildings use inessive/illative: `pankki`→pankissa/pankkiin,
`apteekki`→apteekissa/apteekkiin, `posti`→postissa/postiin, `keskusta`→keskustassa/keskustaan.
"К врачу" idiom = illative `lääkäriin` (NOT lääkärille).

**Verb case government verified:** `käydä` + INESSIVE for visiting (`käyn lääkärissä`,
`käyn kaupassa` — being/visiting), vs `mennä` + ILLATIVE for going-to (`menen pankkiin`).
`sopia` + ALLATIVE for the person ("подходит мне" = `sopii minulle`, not partitive minua).
`tarvita`/`etsiä` take PARTITIVE objects (`tarvitsen rahaa`, `etsin pankkia`).
`maksaa kortilla` = pay BY card → ADESSIVE instrument (not object).

**aikoa + infinitive** = future plan: `Aion mennä lääkäriin` (NOT `aion menen`). Same
infinitive-not-finite trap as haluta/voida.
